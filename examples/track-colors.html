Colors of Track

Colors of features in a track can be specified with a character vector which will be expanded to the same length of track data. The values will be part of the track data and can be modified after the track is constructed.

suppressPackageStartupMessages({
    library(TnT)
    library(GenomicFeatures)
    library(TxDb.Hsapiens.UCSC.hg19.knownGene)
})

R Color Names

This example shows all available color names in R with a block track.

colors <- colors()
data <- GRanges("Unknown", IRanges(seq_along(colors), width = 2), "Color Name" = colors)
track <- BlockTrack(data, color = data$"Color Name", label = NULL)
TnTBoard(track)

Use Color Palettes

R have functions to assist the picking of colors, by which a qualitative value can be mapped to colors, e.g. the rainbow function.

# A helper function
showCols <- function (colors) {
    data <- GRanges("Unknown", IRanges(seq_along(colors), width = 2), "Color" = colors)
    track <- BlockTrack(data, color = data$"Color", label = NULL)
    TnTBoard(track, view.range = range(data))
}
showCols(rainbow(6))
## - Coordinate limit is not specified and seqlength is unknown:
##   automatically set coordinate range to -1..8 ...
showCols(heat.colors(6))
## - Coordinate limit is not specified and seqlength is unknown:
##   automatically set coordinate range to -1..8 ...

Gene Colors

This example shows a gene track that genes have different colors based on their biotypes.

First is to get the data from EnsDb.

library(EnsDb.Hsapiens.v86)
gene <- genes(EnsDb.Hsapiens.v86)

TnT has defined a helper function mapcol to scale qualitative values to colors. We supply the scaled color to the track constructor.

genetrack <- FeatureTrack(gene, tooltip = as.data.frame(gene),
                          color = mapcol(gene$gene_biotype, palette.fun = grDevices::rainbow))
TnTGenome(genetrack, view.range = gene[gene$symbol == "BRCA2"][1] * .5)
